# ?? Telegram Bot для управления чатами 
 
Многофункциональный бот для управления Telegram чатами. 
 
## ? Возможности 
 
- ?? Модерация: бан, мут, варны с автобаном 
- ?? Статистика активности участников 
- ?? Настраиваемые приветствия 
- ?? Автоочистка системных сообщений 
- ?? Пользовательские команды (как Nightbot) 
- ?? Панель управления в личных сообщениях 
- ?? Гибкая система администраторов бота 
 
## ?? Команды 
 
### ?? Администраторы 
 
- `/admin` - панель управления 
- `/ban [причина]` - забанить 
- `/mute [15m|1h|24h|7d]` - замутить 
- `/unmute` - снять мут 
- `/warn [причина]` - выдать варн 
- `/unwarn` - снять варн 
- `/warns` - посмотреть варны 
- `/toggle_clean` - вкл/выкл очистку 
- `/check_perms` - проверить права 
 
### ?? Пользовательские команды 
 
- `/addcmd команда текст` - создать команду 
- `/delcmd команда` - удалить команду 
- `/commands` - список всех команд 
- `!команда` - использовать команду 
 
### ?? Для всех 
 
- `!команда` - выполнение пользовательских команд 
 
## ?? Переменные для команд 
 
- `{name}` - имя пользователя 
- `{user}` - username или имя 
- `{id}` - ID пользователя 
- `{chat_name}` - название чата 
- `{date}` - текущая дата 
- `{time}` - текущее время 
 
## ?? Установка 
 
1. Клонируйте репозиторий: 
   ``` 
   git clone https://github.com/username/telegram-bot.git 
   cd telegram-bot 
   ``` 
 
2. Установите зависимости: 
   ``` 
   pip install -r requirements.txt 
   ``` 
 
3. Создайте файл `.env` и добавьте токен: 
   ``` 
   BOT_TOKEN=ваш_токен_бота 
   ``` 
 
4. Запустите бота: 
   ``` 
   python main.py 
   ```

